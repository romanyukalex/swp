1. Войдите в Админ панель: вашСайт/commenton/panel.php
Перейдите во вкладку Настройки->Соц. сети, сюда будем вставлять ключи выданные соц. сетями.

Далее пошагово

Вконтакте:
- Перейдите по ссылке: https://vk.com/apps?act=manage
- Нажмите "Создать приложение"
- Введите название приложения
- Платформу выберите "Вебсайт"
- В поле "Адрес сайта" введите (https | http)://вашСайт/commenton/authResult.php
- В поле "Базовый домен" введите (https | http)://вашСайт
- Нажмите "Подключить сайт"
- Для подтверждения вы получите код по СМС, введите код в поле и нажмите "Отправить"
- Откроется страница вашего приложения, перейдите в Настройки
- "Состояние" поставьте на "Приложение отключено"
- Скопируйте "ID приложения" и вставьте его в поле "client_id" настроек скрипта
- Скопируйте "Защищённый ключ" и вставьте его в поле "client_secret" настроек скрипта

Одноклассники:
- Перейдите по ссылке: https://ok.ru/dk?st.cmd=appsInfoMyDevList
- Нажмите "Добавить приложение"
- Введите название приложения
- Нажмите "Подключить oAuth"
- В поле "Список разрешённых redirect_uri" введите (https | http)://вашСайт/commenton/authResult.php
- Сохранить
- На e-mail высланы все необходимые ключи, откройте почту
- Скопируйте "Application ID" и вставьте его в поле "client_id" настроек скрипта
- Скопируйте "Публичный ключ приложения" и вставьте его в поле "public_key" настроек скрипта
- Скопируйте "Секретный ключ приложения" и вставьте его в поле "client_secret" настроек скрипта

Faceboock:
- Перейдите по ссылке: https://developers.facebook.com/apps/
- Нажмите "Добавить новое приложение"
- Введите название приложения
- Нажмите "Создайте ID приложения"
- Добавьте продукт "Вход через Facebook"
- Перейдите во вкладку: "Вход через Facebook -> Настройки"
- В поле "Действительные URI перенаправления для OAuth" введите https://вашСайт/commenton/authResult.php?provider=fbAuth (разрешен только https протокол)
- Сохранить
- На странице приложения перейдите во вкладку "Настройки -> Основное"
- В поле "URL-адрес политики конфиденциальности" ведите соответствующий адрес (для работы FB у вас на сайте должна быть страница с политикой конфиденциальности)
- Выберите "Категорию" (нпример: обмен сообщениями)
- Сохранить
- На верхней панели, переключите индикатор в состояние "Вкл"
- Скопируйте "Идентификатор приложения" и вставьте его в поле "client_id" настроек скрипта
- Скопируйте "Секрет приложения" и вставьте его в поле "client_secret" настроек скрипта

Google:
- Перейдите по ссылке: https://console.developers.google.com/iam-admin/projects
- Нажмите "Создать проект"
- Введите название проекта
- Нажмите "Создать"
- Перейдите в проект
- Далее меню в левом верхнем углу - перейдите в раздел  "API и сервисы" > "Учетные данные"
- Перейдите во вкладку "Окно запроса доступа OAuth"
- Введите "Название приложения", в поле "Авторизованные домены" введите ваш домен и сохраните
- Вас перекинет во вклвдку "Учетные данные", здесь - "Создать учетные данные" -> "Идентификатор клиента OAuth"
- Тип приложения выберете "Веб-приложение"
- Введите название
- В поле "Разрешенные URI перенаправления" введите (https | http)://вашСайт/commenton/authResult.php?provider=gAuth
- Создать
- Скопируйте "Идентификатор клиента" и вставьте его в поле "client_id" настроек скрипта
- Скопируйте "Секрет клиента" и вставьте его в поле "client_secret" настроек скрипта

Мой Мир:
- Перейдите по ссылке: http://api.mail.ru/sites/
- Нажмите "Подключить сайт"
- Введите название и Адрес главной страницы
- Следуя инструкции добавьте файл receiver.html себе на сайт и нажмите "Продолжить"/"Пропустить"
- Скопируйте "ID" и вставьте его в поле "client_id" настроек скрипта
- Скопируйте "Секретный ключ" и вставьте его в поле "client_secret" настроек скрипта
- Перейдите в "Мои сайты", управление созданным сайтом
- В поле "Адрес страницы receiver.html" укажите путь к этому файлу, например: (https | http)://вашСайт/receiver.html
- Сохранить

Яндекс:
- Перейдите по ссылке: https://oauth.yandex.ru/
- Нажмите "Зарегистрировать новое приложение"
- Введите название
- В поле "Ссылка на сайт приложения" введите адрес вашего сайта
- Платформы - выбирете "Веб-сервисы"
- В поле "Callback URL" введите (https | http)://вашСайт/commenton/authResult.php?provider=yaAuth
- Доступы* - выберете "API Яндекс.Паспорта" и проставьте галочки
- Сохранить
- Скопируйте "ID" и вставьте его в поле "client_id" настроек скрипта
- Скопируйте "Пароль" и вставьте его в поле "client_secret" настроек скрипта
