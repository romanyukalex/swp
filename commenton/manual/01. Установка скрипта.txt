1. Загрузить папку из архива в корневую директорию вашего сайта

2. Создайте базу данных.

3. Откройте страницу вашСайт.ru/commenton/install.php - следуйте указаниям

4 На страницы, где должны быть комментарии, подключите скрипт, одним из двух вариантов:
4.1 С помощью PHP кода:
<?php include ($_SERVER['DOCUMENT_ROOT'].'/commenton/index.php'); ?>

4.2 С помощью javascript:
- Там, где нужно подключить скрипт, добавить следующий код <div id="commenton_script"></div>
- Подключить библиотеку jQuery, если еще не подключена
- В конде файла, перед тегом </body> подключить файл загрузки <script src="/commenton/js/commenton_load.js"></script>

5. Подключите стили между тегами <head></head>:
<link type="text/css" rel="stylesheet" href="/commenton/style/content_view.css">

6. Не забудьте удалить файл install.php

Почти готово, осталось настроить авторизацию через соц. сети

*Скрипт использует сессии, поэтому они должны быть подключены.
