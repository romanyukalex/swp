Эта папка содержит подключаемые модули системы
Каждый модуль в отдельной папке.
Каждый модуль общается с системой черей запросы на /ajax и использует параметр mod, для идентификации папки
В папке есть файлы:
design.php - html и php-код, который определяет, как будет выглядеть модуль на сайте
ajax.php - обработчик ajax-запросов
Другие файлы, необходимые обработчику ajax или design-модулю. Например some.js - javascript-коды для модуля
