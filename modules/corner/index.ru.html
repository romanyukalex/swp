Easing-функция указывает скорость течения анимации, чтобы сделать анимацию более реалистичной.

Реальные вещи не начинают двигаться мгновенно и с постоянной скоростью. Если мы открываем ящик стола, то сначала ускоряем его, а на второй половине пути — тормозим. Если что-то упало, то оно сначала летит всё быстрее и быстрее, а ударившись о пол — слегка подпрыгивает обратно.

Этот сайт поможет каждый раз подобрать нужную easing-функцию.

JavaScript

Проще всего рисовать анимации по easing-функции через jQuery, добавив jQuery Easing Plugin. После этого надо лишь указать имя функции в методе .animate третьим аргументом или через ключ easing.

div.animate({ top: '-=100px' }, 600, 'имя easing-функции', function () { … })

Sass

В Sass/SCSS анимации описывать гораздо проще — Compass избавляет от префиксов перед свойствами transition и animation, а расширение Compass Ceaser позволит указывать easing-функцию просто по имени, не думая о кривых Безье.

div
  @include transition(all 600ms ceaser('имя easing-функции'))

К сожалению, такую easing-функцию нельзя задать с помощью CSS. Используйте JavaScript или специальные @keyframes в CSS Animation.

CSS

В CSS свойства transition и animation позволяют указывать easing-функцию. К сожалению, доступны не все функции и чаще всего указывать их надо через кривую Безье.

Выберите easing-функцию, чтобы увидеть, как её описать через кривую Безье.

div {
  -webkit-transition: all 600ms описание easing-функции;
  -moz-transition:    all 600ms описание easing-функции;
  -ms-transition:     all 600ms описание easing-функции;
  -o-transition:      all 600ms описание easing-функции;
  transition:         all 600ms описание easing-функции; }

К сожалению, такую easing-функцию нельзя задать с помощью CSS. Используйте JavaScript или специальные @keyframes в CSS Animation.
